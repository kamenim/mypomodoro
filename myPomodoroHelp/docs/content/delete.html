Delete a Task

You can only delete a single task from the Activity List. There are three ways to open the Activity List.

  • Click the View Menu and select Activity List in the Menu-Based Navigation.
  • Press Alt + A
  • Click the Activity Icon from the Image-Based Navigation.

Then all you have to do is select the task that you want to delete and click the Delete Button. The task is then permanently deleted from the database. See also Activity List, Menu-Based Navigation, Image-Based Navigation, and Keyboard Shortcuts.
