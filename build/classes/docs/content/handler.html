Handle Interruptions

During a Pomodoro, Interruptions must be handled. Interruptions are handled within the ToDo List during a Pomodoro. When a current Pomodoro is interrupted, the user can click on the Interruptions Tab in the ToDo List and create an Unplanned Task much like the way you Create a Task. That Unplanned Task is moved to the Activity List so that the user can get back to his/her current Pomodoro. Whenever an Unplanned Task is created under a Pomodoro, that task is marked with an Interruption. See also Unplanned Task and ToDo List.
