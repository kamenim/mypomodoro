Image-Based Navigation

The Image-Based Navigation consists of the following menus:

  • Create – Allows you to create a new task. See also Create panel, Menu-Based Navigation, and Keyboard Shortcuts.
  • Activity List – Same as the Activity Inventory List in the Pomodoro Technique™. See also Activity List, Menu-Based Navigation, and Keyboard Shortcuts.
  • Manager – Allows the user to create a ToDo List from the Activity List as well as set the order/priority of each task. See also Manager panel, Menu-Based Navigation, and Keyboard Shortcuts.
  • ToDo List – Same as the ToDo Today Sheet in the Pomodoro Technique™, in addition to the Pomodoro Timer. See also ToDo List, Menu-Based Navigation, and Keyboard Shortcuts.
  • Report List - Same as the ToDo Today Sheet in the Pomodoro Technique™. See also Report List, Menu-Based Navigation, and Keyboard Shortcuts.
