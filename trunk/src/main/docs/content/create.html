Creating Tasks

Creating a task is actually the first step any user must do when they first start using this application or have no remaining tasks in both the Activity List and ToDo List. In order to create a task you must open the Create panel. There are three ways to do this:

  • Click the File Menu and select New Activity in the Menu-Based Navigation.
  • Press Alt + N
  • Click the Create Icon from the Image-Based Navigation.

*Optional method

  • Go to the Activity List (Alt + A) and select the Create Task Tab

Then all you have to do is fill out the form and press the Save Button. Your task is automatically added to the Activity List. Keep adding tasks until you are satisfied. See also Create panel, Menu-Based Navigation, Image-Based Navigation, and Keyboard Shortcuts.
