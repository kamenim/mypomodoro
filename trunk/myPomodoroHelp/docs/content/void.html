Void a Task

Void is for the cases when an interruption takes priority over that of the current Pomodoro. When a user voids a task that task is marked as voided, the timer resets and stops, and the task is removed from the ToDo List. To void a task you must be in the ToDo List panel. There are three ways to open the ToDo List:

  • Click the View Menu and select ToDo List in the Menu-Based Navigation.
  • Press Alt + T
  • Click the ToDo Icon from the Image-Based Navigation.

Then all you have to do is select the task that you are voiding and select the Void Button. See ToDo List.
