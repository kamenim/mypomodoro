ToDo List

This panel displays the current ToDo List (ToDo Today Sheet) and allows the user to perform the methods 2-7 as specified in About the Pomodoro Technique. It provides the timer feature, marking interruptions on a current task by creating unplanned tasks that are added to the Activity List, and providing a feature to void tasks. When the task is completed or voided it is moved to the Report List with the correct markups and actual number of Pomodoroâ€™s for that task.
