About the Pomodoro Technique™

The Pomodoro Technique™ is a time management technique created by Francesco Cirillo for a more productive way to work and study. This technique eliminates the concept of time by using pure units of work, called Pomodoro’s, without any interruptions. A traditional Pomodoro is 30 minutes long: 25 minutes of uninterrupted work plus a 5-minute break. Typical materials used in this technique include: a kitchen timer, a ToDo Today Sheet, an Activity Inventory Sheet, and a Records Sheet. The basic methods used to follow this technique include:

  1. Choose a task to be accomplished from the Activity Inventory Sheet
  2. Set the timer to 25 minutes
  3. Work on the task until the timer rings
  4. Put a check on the ToDo Today Sheet
  5. Take a short break
  6. Every 4 Pomodoro's take a longer break
  7. When the task is completed cross it off the ToDo Today Sheet
  8. At the end of the day record all completed tasks in the Records Sheet
