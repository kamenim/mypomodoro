Menu-Based Navigation

The Menu-Based Navigation consists of the following menus and submenus:

  • File
    • Preferences – Allows you to set the timer for the Pomodoro Length (Default: 25 min), Short Break (Default: 5 min), and Long Break (Default: 45 min) of the Digital Pomodoro Timer located in the ToDo List. See also Keyboard Shortcuts.
    • New Activity – Allows you to create a new task. See also Create panel, Image-Based Navigation, and Keyboard Shortcuts.
    • Exit – Closes the Application
  • View
    • Activity List – Same as the Activity Inventory List in the Pomodoro Technique™. See also Activity List, Image-Based Navigation, and Keyboard Shortcuts.
    • Manager – Allows the user to create a ToDo List from the Activity List as well as set the order/priority of each task. See also Manager panel, Image-Based Navigation, and Keyboard Shortcuts.
    • ToDo List – Same as the ToDo Today Sheet in the Pomodoro Technique™, in addition to the Pomodoro Timer. See also ToDo List, Image-Based Navigation, and Keyboard Shortcuts.
    • Report List - Same as the Records Sheet in the Pomodoro Technique™. See also Report List, Image-Based Navigation, and Keyboard Shortcuts.
  • Data
    • Clear All Data – Allows user to delete all data, but does not delete the database “pomodoro.db.” See also Deleting All Data.
    • Populate Test Data – Allows first-time users to generate random data to learn how to use this software in conjunction with the Pomodoro Technique™.
  • Help
    • Help System – serves as the user manual for this software. See also Keyboard Shortcuts, and Get Help.
