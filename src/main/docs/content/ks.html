Keyboard Shortcuts

We believe we have made some pretty easy to remember Keyboard Shortcuts. The current keyboard shortcuts are written below:

  • Alt + P for Preferences
  • Alt + N for New Activity/Create Task
  • Alt + A for Activity List
  • Alt + R for Report List
  • Alt + M for Manager
  • Alt + T for ToDo List
  • Alt + H for Help System

As you may have notice all of our shortcuts start with the Alt-Key and end with the letter that corresponds to the feature being requested. Shortcut hints are also given in the drop down menus.
