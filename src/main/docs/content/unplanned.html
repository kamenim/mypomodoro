Unplanned Tasks

An Unplanned Task is a task that was created as an Interruption to a Pomodoro in the ToDo List. To create an interruption you must be in the ToDo List. There are three ways to open the ToDo List:

  • Click the View Menu and select ToDo List in the Menu-Based Navigation.
  • Press Alt + T
  • Click the ToDo Icon from the Image-Based Navigation.

Then all you do is click the Interruptions Tab, fill out the form, and click saves. The task is placed in the Activity List and is marked as an unplanned task.

See also Interruption and ToDo List.
