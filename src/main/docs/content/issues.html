myPomodoro - Java Desktop Application

Help System

Known Issues

  • Timer runs one full break before stopping (not sure if an issue based on the Pomodoro Technique because “Once a Pomodoro Starts you cannot stop it” (Pomodoro= 30 min = 25 min Uninterrupted Work + 5 min Break).
  • Currently there is no means to edit any data, you must delete that task you would like to edit and add a new task.
  • No ticking noise option on the timer.
  • Little notification between Pomodoro time and breaks.
  • If you have something selected in any list and clear all data, the data from the last thing selected remains in the details area for that panel.
  • On Windows XP, the .ico file does not render properly.
  • At random times, if you populate test data and go to the Activity List, you will see two date columns. The next time the panel refreshes it’s gone. Does not happen often.
  • The timer shifts whenever a 1 is displayed, this is due to a difference in font width of the number 1 being different from the others.
  • No ability for the user to develop sets within the application. The user can set up sets by using their estimates and only transferring tasks one set at a time into the ToDo List.
